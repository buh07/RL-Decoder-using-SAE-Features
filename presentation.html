RL-Decoder with SAE Features

This project tests whether internal arithmetic representations can be turned into a meaningful Chain-of-Thought (CoT) faithfulness auditor, not just an answer predictor.

The GPT-2 Medium branch is now closed for Phase 7: strong mechanistic arithmetic evidence, and post-closure stress tests still leave Track C as a negative result for faithfulness discrimination.

Project state: Phases 1-6 complete; Phase 7 GPT-2 closed; Qwen inquiry open.
Final GPT-2 Phase 7 config: text=0.50, latent=0.50, structural penalties on.
Status snapshot date: March 7, 2026.

Motivation And Research Question

Motivation

  • Models can output plausible reasoning that does not reflect internal computation.
  • Answer accuracy cannot certify reasoning faithfulness.
  • Mechanistic evidence is needed to separate genuine faithfulness from rationalization.

Core question (Phase 7)

  • Can a three-track auditor (text, latent, causal) discriminate faithful vs unfaithful CoT?
  • Can it do so under leakage-safe split policy and strict gate criteria?
  • Does mechanistic Track C add real marginal value?

Current answer (GPT-2)

  • Track C does not clear closure-grade relevance for GPT-2 under this protocol.
  • Two-track auditing is the final GPT-2 output.
  • Qwen is a separate test of whether this is model-specific.

Three Publishable GPT-2 Results

Result 1: Mechanistic arithmetic is real

  • Arithmetic state is strongly decodable from hidden representations.
  • Subspace interventions produce positive causal movement in output behavior.

Result 2: Mechanism does not imply CoT faithfulness

  • Track C reformulations did not provide robust faithful/unfaithful discrimination for GPT-2.
  • This shows a measurable readout-versus-faithfulness gap.

Result 3: Text+structure still works as a detector

  • Fair Option B profile comparison yields composite AUROC around 0.836-0.838.
  • Useful as a practical detector, but not a mechanistic faithfulness guarantee.
